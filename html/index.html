關於

沃羅諾伊圖(Voronoi Diagram)為一個演算法可以幫助我們空間分割，可應用於幾何、建築地理學 等。其特色為圖上任意點落於哪個區域(黑色線段所圍區域)中則與該點距離最近的紅點必也在 該區域中。

軟體規格書

輸出與輸入(資料)規格

輸入資料要求為須輸入測試點數，再逐一輸入點的x座標與y座標，輸入點數為0時表示測試結束。 輸出會依照不同的圖片輸出不同的檔案，在每行前面加上P或E表示為point(點)或edge(邊)。

功能規格與介面規格

主介面如下圖
read file : 讀取原始檔案
read file(old) : 讀取輸出檔案
next set : 顯示下個點集合
write file : 將目前的點與線輸出
play : 將voronoi diagram一次性完成
step by step : 依步驟完成voronoi diagram
clear : 將全畫面清空

軟體測試規劃書

本程式為使用python設計，並用python套件tkinter做為視窗介面工具。 其中voronoi diagram的以遞迴方式解，而遞迴最小單位為2點。 對於任何大於2點的點集合會將點結合分隔成兩半直至點數為2為止。 再將較小點集合的voronoi diagram做合併直至原點和為止。
